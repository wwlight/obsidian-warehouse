## CSS中渐变角度的概念
- 在 CSS 渐变中，角度的定义遵循以下规则：
	- 0 度是从下往上
	- 90 度是从左往右
	- 180 度是从上往下
	- 270 度（或 -90 度）是从右往左

## 和数学圆中角度的区别
- 起始位置不同：
    - CSS 渐变：0° 指向上方（12点钟方向）
    - 数学角度：0° 指向右方（3点钟方向）
- 旋转方向不同：
    - CSS 渐变：顺时针旋转
    - 数学角度：逆时针旋转
- 常用角度对应关系：
```
	 CSS渐变    数学角度
	 0°    →   90°
	 90°   →   0°
	 180°  →   270°
	 270°  →   180°
```
- 实际应用场景：
	- CSS渐变角度主要用于设计和用户界面，更符合视觉直觉（从上到下更自然）
	- 数学角度用于数学计算和科学场景，符合笛卡尔坐标系统
## 转换方式
```js
// CSS角度转数学角度
const cssToMath = (cssAngle) => (90 - cssAngle + 360) % 360;

// 数学角度转CSS角度
const mathToCss = (mathAngle) => (90 - mathAngle + 360) % 360;
```
